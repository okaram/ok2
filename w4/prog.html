Programming

  • ctrl-shift-j in chrome opens the javascript console
  • you type javascript code and it gets executed immediately when you press enter
  • Can use as a calculator; type: 3+2 (then enter)
  • Can define variables var x=3; and use them later x+3
  • Can define functions: function add(a,b) { return a+b;} and use them later add(3,4)
  • Can use it to change your document !! document.bgColor='lightblue'
  • Can use document.getElementById to get a specific piece of your document, and modify it img==document.getElementById('theImg') will get you the image there, and store it in the variable img
  • And can modify those elements: img.width=200
  • Can create new elements: snd=document.createElement('audio'); then snd.src='abc.ogg' and snd.play()
  • Can use the <script> tag to add javascript to a page, either directly (embed) or point to another file with src <script type="text/javascript"></script>
